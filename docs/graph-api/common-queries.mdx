# Consultas Comuns de IAM na Graph API

Este guia fornece exemplos práticos de operações de gerenciamento de identidade e acesso usando a Microsoft Graph API.

## Consulta de Usuários

### Obter lista de usuários

```http
GET https://graph.microsoft.com/v1.0/users
```

### Filtrar usuários

```http
GET https://graph.microsoft.com/v1.0/users?$filter=startswith(displayName,'John')
```

### Obter usuário específico

```http
GET https://graph.microsoft.com/v1.0/users/john.doe@contoso.com
```

## Gerenciamento de Grupos

### Listar todos os grupos

```http
GET https://graph.microsoft.com/v1.0/groups
```

### Criar um grupo

```http
POST https://graph.microsoft.com/v1.0/groups
Content-Type: application/json

{
  "displayName": "Grupo de Segurança IAM",
  "mailEnabled": false,
  "securityEnabled": true,
  "description": "Grupo para gerenciamento de acesso IAM"
}
```

### Adicionar membro a um grupo

```http
POST https://graph.microsoft.com/v1.0/groups/{group-id}/members/$ref
Content-Type: application/json

{
  "@odata.id": "https://graph.microsoft.com/v1.0/directoryObjects/{user-id}"
}
```

## Consultas de Acesso Condicional

### Listar políticas de acesso condicional

```http
GET https://graph.microsoft.com/v1.0/identity/conditionalAccess/policies
```
