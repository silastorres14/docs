# Introdução à Microsoft Graph API

O Microsoft Graph é a porta de entrada para dados e inteligência na plataforma Microsoft 365. Ele fornece um modelo de programação unificado para acesso a uma grande quantidade de recursos, relacionamentos e inteligência.

## O que é o Microsoft Graph?

Microsoft Graph é um gateway via RESTful API que permite acesso a:

- Dados de usuários, grupos e dispositivos no Entra ID
- Serviços Office 365 como Exchange, SharePoint, OneDrive, Teams
- Serviços de segurança e identidade
- Dados do Windows 10/11 em dispositivos gerenciados

## Por que usar o Microsoft Graph?

- **API Unificada** - Um único endpoint (https://graph.microsoft.com) para todos os serviços
- **Relacionamentos ricos** - Navegação entre entidades e seus relacionamentos
- **Insights e inteligência** - IA e ML aplicados aos dados
- **SDKs para diversas plataformas** - .NET, JavaScript, Java, Python e mais
