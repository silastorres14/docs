# Configurando o SDK do PowerShell para Microsoft Graph

O módulo Microsoft.Graph para PowerShell é um conjunto de cmdlets baseado em REST para gerenciar recursos do Microsoft Entra ID e outros serviços da Microsoft 365.

## Instalação do Módulo

Você pode instalar o módulo completo ou instalar apenas os submódulos que precisa:

```powershell
# Instalar o módulo completo
Install-Module Microsoft.Graph -Scope CurrentUser

# OU Instalar apenas submódulos específicos
Install-Module Microsoft.Graph.Authentication -Scope CurrentUser
Install-Module Microsoft.Graph.Users -Scope CurrentUser
Install-Module Microsoft.Graph.Groups -Scope CurrentUser
Install-Module Microsoft.Graph.Identity.DirectoryManagement -Scope CurrentUser
```

## Autenticação com o Graph

Existem várias maneiras de se autenticar ao Microsoft Graph usando PowerShell:

```powershell
# Método 1: Autenticação interativa (com permissões delegadas)
Connect-MgGraph -Scopes "User.Read.All", "Group.ReadWrite.All"

# Método 2: Autenticação de aplicativo (permissões de aplicativo)
Connect-MgGraph -ClientId "YOUR_APP_ID" -TenantId "YOUR_TENANT_ID" -CertificateThumbprint "YOUR_CERT_THUMBPRINT"

# Método 3: Usando um arquivo de certificado
Connect-MgGraph -ClientId "YOUR_APP_ID" -TenantId "YOUR_TENANT_ID" -Certificate $cert
```

## Verificando a Conexão

Após a autenticação, você pode verificar se está conectado e quais permissões estão disponíveis:

```powershell
# Verificar status de conexão
Get-MgContext

# Verificar permissões concedidas
(Get-MgContext).Scopes
```
