# Visão Geral e Introdução

Bem-vindo ao Guia de Gestão de Identidade Microsoft. Este recurso fornece informações abrangentes sobre como implementar e gerenciar soluções de Identidade e Acesso usando tecnologias Microsoft.

## O que você encontrará nesta documentação

Esta documentação cobre três áreas principais:

- **Microsoft Entra ID** - Conceitos, configuração e gerenciamento
- **Microsoft Graph API** - Como usar a API para automatizar tarefas de IAM
- **PowerShell para IAM** - Scripts e comandos para gerenciamento de identidade

## Gestão de Identidade e Acesso

A Gestão de Identidade e Acesso (IAM) é fundamental para a segurança moderna, permitindo:

- Controlar quem tem acesso aos seus recursos
- O que podem fazer com esse acesso
- Monitorar como o acesso é utilizado
