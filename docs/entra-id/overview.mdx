# Visão Geral do Microsoft Entra ID

O Microsoft Entra ID (anteriormente Azure Active Directory) é o serviço de gerenciamento de identidade e acesso baseado em nuvem da Microsoft, que ajuda seus funcionários a acessar recursos externos e internos.

## O que é o Microsoft Entra ID?

O Entra ID é um serviço de diretório na nuvem que fornece:

- Gerenciamento de identidades
- Autenticação 
- Controle de acesso baseado em políticas
- Colaboração B2B e B2C
- Integração com milhares de aplicativos SaaS

## Arquitetura Básica

O Entra ID permite que você gerencie o acesso aos seus aplicativos e recursos com base em regras de negócios centralizadas e políticas de segurança.
