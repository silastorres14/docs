# Gerenciamento de Usuários e Grupos no Entra ID

Uma das principais funcionalidades do Microsoft Entra ID é o gerenciamento de usuários e grupos, permitindo organizar o acesso a recursos de forma eficiente.

## Usuários

Os usuários representam identidades individuais dentro do seu diretório que podem:

- Fazer login em aplicativos integrados ao Entra ID
- Acessar recursos protegidos 
- Ter permissões atribuídas individualmente ou através de grupos

## Grupos

Os grupos no Entra ID permitem:

- Organização lógica de usuários
- Atribuição de acesso em massa
- Delegação de permissões administrativas
- Gerenciamento simplificado de licenças

### Tipos de Grupos

- **Grupos de Segurança** - Usados para gerenciar acesso a recursos
- **Grupos Microsoft 365** - Colaboração com serviços como Teams, SharePoint
- **Grupos Dinâmicos** - Associação baseada em atributos e regras
