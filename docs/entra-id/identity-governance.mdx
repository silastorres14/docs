# Governança de Identidade (Identity Governance)

A Governança de Identidade no Microsoft Entra ID permite que as organizações equilibrem segurança e produtividade, garantindo que as pessoas certas tenham o acesso certo aos recursos certos.

## Componentes principais

A Governança de Identidade no Entra ID inclui:

- **Gerenciamento de direitos (Entitlement Management)** - Gerencia o acesso a grupos, aplicativos e sites
- **Revisões de acesso (Access Reviews)** - Permite revisões periódicas de acesso
- **Gerenciamento de acesso privilegiado (PIM)** - Gerencia identidades com acesso elevado
- **Gerenciamento do ciclo de vida** - Provisionamento e desprovisionamento automatizados

## Benefícios da Governança de Identidade

- **Gestão de riscos aprimorada** - Identificação e mitigação de riscos de acesso
- **Conformidade melhorada** - Atendimento a requisitos regulatórios
- **Redução de custos** - Automação de processos manuais
- **Produtividade** - Acesso aos recursos certos no momento certo
