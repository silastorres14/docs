## Projetos Disponíveis

### [Sistema de Monitorização de Credenciais](./app-credential-monitoring)

Uma solução completa para monitorizar e gerir credenciais de aplicações no Microsoft Entra ID, ajudando a prevenir interrupções de serviço causadas por segredos ou certificados expirados.

**Tecnologias:** PowerShell, Microsoft Graph API, Power Automate, Power BI

**Nível de Complexidade:** Médio
